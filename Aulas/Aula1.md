# 🧠 Pensamento Computacional

O **pensamento computacional** é um processo de **resolução de problemas** que envolve estruturar soluções de forma que um computador possa compreender e executar.

Ele é composto por quatro pilares principais:

---

# 1️⃣ Decomposição

Consiste em **dividir um problema complexo em partes menores**, facilitando sua resolução.

### Exemplo

Um projeto de software pode ser dividido em:

- Criar interface
- Programar banco de dados
- Implementar funcionalidades
- Realizar testes

Outro exemplo apresentado é o **planejamento de um hackathon**, dividido em etapas como:

- definição do tema
- definição do público
- organização logística
- divulgação
- avaliação dos projetos

---

# 2️⃣ Reconhecimento de Padrões

É identificar **semelhanças entre problemas ou soluções**.

### Exemplo

Na programação, quando percebemos tarefas repetidas no código, podemos criar:

- funções
- classes
- métodos reutilizáveis

Outro exemplo é quando características comuns são agrupadas em uma **classe pai (superclasse)** utilizando **herança**.

---

# 3️⃣ Abstração

A abstração consiste em **focar nas informações importantes e ignorar detalhes desnecessários**.

### Exemplo

Em um sistema de gerenciamento de tarefas, os elementos essenciais são:

- tarefa
- responsável
- data de entrega

Detalhes irrelevantes como:

- cor da mesa
- número de canetas

não influenciam no funcionamento do sistema.

---

# 4️⃣ Algoritmos

Algoritmos são **sequências de passos para resolver um problema**.

### Exemplo

Algoritmo para fazer café:

1. Colocar água na chaleira  
2. Aquecer até ferver  
3. Colocar café no filtro  
4. Despejar água quente sobre o café  
5. Servir

# 🎂 Desafio: Bolo Maluco em JavaScript

Criar um algoritmo em **JavaScript** que monte um bolo.

### Requisitos

- 3 camadas de massa
- 3 recheios
- 1 cobertura final

---

## Passos

1. Abrir o **VS Code**  
2. Criar um arquivo chamado `bolo.js`  
3. Criar variáveis para massa, recheio e cobertura  
4. Usar `console.log()` para mostrar cada etapa  

---

## 💻 Exemplo

```javascript
let massa1 = "massa de chocolate"
let massa2 = "massa de baunilha"
let massa3 = "massa de morango"

let recheio1 = "recheio de nozes"
let recheio2 = "recheio de ninho"
let recheio3 = "recheio de nutella"

let cobertura = "cobertura de chocolate"

console.log("Montando o bolo...")

console.log("Camada 1:", massa1)
console.log("Recheio:", recheio1)

console.log("Camada 2:", massa2)
console.log("Recheio:", recheio2)

console.log("Camada 3:", massa3)
console.log("Recheio:", recheio3)

console.log("Cobertura:", cobertura)

```
